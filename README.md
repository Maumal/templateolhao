# PostFácil

## Crie Posts de Promoção Profissionais em 2 Cliques

---

## Como Começar (30 segundos)

1. Abra o arquivo **index.html** no seu navegador (duplo-clique ou arraste para o Chrome/Firefox)
2. Escolha um template na seção "Templates Rápidos"
3. Carregue a imagem do seu produto (botão "Carregar Imagem")
4. Ajuste o nome, preços e cores se quiser
5. Clique em **Baixar** para salvar sua imagem

---

## Funcionalidades Incluídas

- 12 Templates Prontos (8 padrão + 4 sazonais)
- Modo Escuro/Claro
- Exportar PNG, JPG ou WebP
- Salvar Favoritos (seus estilos preferidos)
- 10 Tamanhos de Post (Instagram, TikTok, Pinterest...)
- Gradientes no fundo
- Borda decorativa
- Sombra no produto
- Copiar para área de transferência

---

## Atalhos de Teclado

| Atalho | Ação |
|--------|------|
| Ctrl + S | Baixar imagem |
| Ctrl + D | Salvar como favorito |
| Ctrl + L | Carregar último post |
| Ctrl + 1/2/3/4 | Aplicar templates rápido |

---

## Dicas Pro

- Use imagens PNG com fundo transparente para melhores resultados
- Salve seus estilos como "Favoritos" para usar depois
- Use "Exportar JSON" para fazer backup das suas configurações
